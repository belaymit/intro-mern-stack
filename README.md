### Prerequisites

In order to run this project you need:
>- A browser that supports html5
>- Git 
>- A GitHub account
>- Node.js

>- Basic understanding of the following:
>- HTM5/CSS3
>- ES6
>- webpack
>- JSX
>- ReactJS

### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git git@github.com:belaymit/intro-mern-stack.git
  
```

### Install

Install this project with:


```sh
  cd intro-mern-stack
  npm install
```
